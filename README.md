# ESP32-Tamagotchi
ESP32 Tamagotchi Pet

This project recreates the Tamagotchi toy using an ESP32 on an Altium PCB. The Tamagotchi toy should have these functions: 
Upon initially activating the Tamagotchi, an egg will appear on screen which hatches into a pet. 
The Tamagotchi will have a hunger meter and a happy meter. The age of the pet can be checked as well.
The pet can be fed to raise its hunger meter. If the needs of the pet are unfulfilled for too long, the happy meter will go down.
If the hunger and happy meter stay low for too long, the pet will die and the player must restart with a new pet.
